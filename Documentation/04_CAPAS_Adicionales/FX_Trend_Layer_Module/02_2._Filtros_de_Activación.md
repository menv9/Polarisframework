## 2. Filtros de Activación

Tres condiciones simultáneas para que un par sea elegible para Capa 2:

```text
FILTRO 1 — Señal macro mínima
  |Señal_FX(par)| ≥ 1.0σ (convicción mínima media)
  La dirección de Capa 2 = dirección de Capa 1.

FILTRO 2 — Régimen de volatilidad apto
  ATR(14) entre P30 y P80 de su distribución 5Y
  Vol < P30 → mercado lateral, breakouts falsos
  Vol > P80 → demasiado ruido para horizonte corto

FILTRO 3 — Liquidez de horario
  Apertura solo en sesión London o NY (o overlap)
  Stops solo durante sesión líquida
  Sesión Asia: monitoreo solo, no entries

Si los 3 filtros NO se cumplen → par no elegible para Capa 2 ese día.
```
