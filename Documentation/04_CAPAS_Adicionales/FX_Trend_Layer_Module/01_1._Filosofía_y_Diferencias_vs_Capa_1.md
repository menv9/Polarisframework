## 1. Filosofía y Diferencias vs Capa 1

### 1.1 Por qué una capa adicional

```text
PROBLEMA de Capa 1 sola:
  - Horizonte 1-3 meses → 2-4 trades/mes típicos
  - Largos períodos sin trade abierto (semanas planas)
  - No captura movimientos cíclicos cortos dentro de la tendencia macro
  - Subutiliza el análisis macro (mismo trabajo, una sola fuente de P&L)

SOLUCIÓN — Capa 2 sobre la misma señal macro:
  - Horizonte 1-2 semanas → 6-10 trades/mes
  - Aumenta turnover sin contradecir tesis macro
  - Captura momentum técnico dentro de la tendencia macro
  - Diversifica fuentes de P&L (señal macro genera ambas capas)
```

### 1.2 Diferencias estructurales con Capa 1

| Dimensión | Capa 1 (Macro) | Capa 2 (Trend) |
|---|---|---|
| Horizonte | 1-3 meses | 1-2 semanas |
| Frecuencia | 2-4 trades/mes | 6-10 trades/mes |
| Risk/trade | 1.0% capital | 0.5% capital |
| Max simultáneos | 3 | 3 |
| Driver de entrada | Señal macro fundamental | Setup técnico (filtrado por macro) |
| Stop ATR multiplier | k = 2.0-3.0 | k = 1.5-2.0 |
| Target (R) | 2-3R | 1.5-2R |
| Re-evaluación | Mensual | Semanal |
| Actualización β | Rolling 60m | Kalman Filter |
| Holding mínimo | 5 días | 2-3 días |
| Holding máximo | 26 semanas (largo) | 3 semanas |

### 1.3 La regla de oro: nunca contradecir

```text
Caso 1 — Señal macro alineada:
  Capa 1: Señal_FX(EUR/USD) = +1.5σ → long EUR/USD
  Capa 2: setup técnico de breakout alcista → long EUR/USD ✓ EJECUTAR
  
Caso 2 — Señal macro contraria:
  Capa 1: Señal_FX(EUR/USD) = −1.2σ → short EUR/USD
  Capa 2: setup técnico de pin bar alcista → ¿long? ✗ NO EJECUTAR
  Capa 2 solo opera shorts en EUR/USD mientras Capa 1 sea bajista.

Caso 3 — Sin señal macro:
  Capa 1: |Señal_FX(EUR/USD)| < 1σ → sin convicción
  Capa 2: setup técnico válido → ¿operar? ✗ NO EJECUTAR
  Sin macro de fondo, Capa 2 no opera ese par.

Caso 4 — Capa 1 se invierte mientras Capa 2 abierto:
  Capa 1 flips de long a short EUR/USD.
  Capa 2 tiene long EUR/USD abierto.
  → Cerrar Capa 2 inmediatamente (al open siguiente sesión).
```
