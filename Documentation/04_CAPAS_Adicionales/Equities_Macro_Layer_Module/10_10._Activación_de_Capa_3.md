## 10. Activación de Capa 3

```text
NO activar Capa 3 hasta que:
  ✓ Capa 1 con ≥ 18 meses operativa estable
  ✓ Capa 1 con Sharpe rolling 12m ≥ 0.6
  ✓ Capa 2 (si activa) con ≥ 12 meses estable
  ✓ Disciplina demostrada en ambas capas anteriores
  ✓ Capital suficiente para diversificar (mín 25k USD por la naturaleza de equities)

Año 1:    Solo Capa 1 (FX macro)
Año 2:    + Capa 2 (FX trend) - opcional
Año 3:    + Capa 3 Fase 1 (ETFs sectoriales solo)
Año 4+:   + Capa 3 Fase 2 (large caps macro-sensibles)

Lanzar Capa 3 antes de Capa 1 estable: error sistémico.
La Capa 3 reutiliza el motor macro; si el motor no funciona, Capa 3 tampoco.
```
