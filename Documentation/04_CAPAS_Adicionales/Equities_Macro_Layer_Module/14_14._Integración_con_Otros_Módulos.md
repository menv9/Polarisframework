## 14. Integración con Otros Módulos

```text
INPUT:
  ← Capa 1 (Endo + Exo + WV): z-scores macro disponibles
  ← Datos equities: precios, ATR, volumen, earnings calendar
  ← Datos broker: comisiones, borrow rates, dividend dates

OUTPUT:
  → Risk Management: Pos_C3 con sizing equities
  → Execution: orden a broker equities (puede ser distinto del FX broker)
  → Self-Awareness: tracking SEPARADO de Capa 1 y Capa 2
      - Sharpe Capa 3 vs Capa 1 (¿añade valor?)
      - Atribución por instrumento y por driver
      - Correlación P&L Capa 3 con Capa 1 y 2 (debe ser < 0.7 ideal)
      - Performance ETFs vs single stocks

NO COMPARTE:
  - Broker (puede ser otro broker para equities, IB unificada o segregada)
  - Margen / cuenta (cuenta separada recomendada para tax tracking)
  - Calendario blackouts (FOMC sí afecta, NFP menos a single stocks)
```
