## 13. Cómo usar este framework

### Si eres principiante en este sistema
```
1. Leer FX_Trading_Framework.md (orquestador, 15 min)
2. Leer este documento (FX_Framework_Schema.md) para tener mapa
3. Profundizar en FX_Endogenous_Module.md §1-§4 (corazón del sistema)
4. Hojear los demás módulos para conocer alcance
5. Releer cada módulo en detalle según necesidad operativa
```

### Si ya conoces el sistema y vas a operar
```
1. Verificar checklist pre-trade en cada nivel:
   - Endo §15.7 (convicción)
   - Timing §11 (setup técnico)
   - Risk §8 (sizing y stops)
   - Execution §11 (ejecución viable)
2. Aplicar pipeline §6 Timing (5 checks secuenciales)
3. Si pasa todos → ejecutar
4. Logging obligatorio en journal (SA §6)
```

### Si vas a calibrar o implementar técnicamente
```
1. Leer Endo §17 (anti-espuria) ANTES de cualquier regresión
2. Aplicar tests 1-5 a cada driver candidato
3. Calibrar β con §9 + walk-forward §16.2
4. Definir capa de actualización (§18: anual / rolling / Kalman)
5. Implementar arranque gradual §11 Risk (Tramo 1 = 20%)
6. Backtest formal con métricas mínimas §9.4
7. Ver Implementation Stack §1-16 para stack técnico (Python, infra, deployment)
8. Seguir Fases A-E del Implementation §12 (16-25 semanas hasta live)
```

### Si vas a expandir a sistema multi-capa
```
PRE-REQUISITO: Capa 1 estable mínimo 12 meses con Sharpe ≥ 0.5

Para añadir Capa 2 (Trend FX 1-2 semanas):
  1. Leer Trend Layer §1-13 completo
  2. Verificar pre-requisitos §11 Trend Layer
  3. Implementar setups C2-1, C2-2, C2-3 (§3)
  4. Activar β con Kalman Filter (§6)
  5. Tracking SEPARADO en Self-Awareness obligatorio
  6. Sizing reducido inicial (50% durante 6 meses)

Para añadir Capa 3 (Equities Macro):
  1. Leer Equities Macro Layer §1-14 completo
  2. Verificar pre-requisitos §10 Equities (≥18m Capa 1 estable)
  3. EMPEZAR por ETFs sectoriales (§2.1 Fase 1)
  4. Mapeo driver → instrumento (§3)
  5. Score_Equity por instrumento (§4)
  6. Single stocks SOLO en año 4+ tras dominar ETFs
```

### Si vas a operar Emerging Markets
```
PRE-REQUISITO: Capa 1 G10 estable mínimo 18 meses + capital ≥ 50k USD

Pasos:
  1. Leer EM Module §1-16 completo
  2. Identificar TIER de cada divisa candidata (§1.1)
   → No operar TIER 3 (pegged) direccional ni TIER 4 (hyperinflation)
  3. Calibrar β con drivers EM específicos (CDS, EMBI, DXY peso alto)
  4. Sizing reducido obligatorio (0.5% RPT vs 1.0% G10)
  5. Convicción mínima 1.8σ (vs 1.0σ G10) por costes mayores
  6. Empezar por las más líquidas: MXN, ZAR
  7. Acceso a NDF si quieres KRW, INR, IDR (broker tier-1 requerido)
  8. Tracking SEPARADO de G10 en Self-Awareness
```

### Si vas a planificar el fiscal
```
1. Leer Tax/Fiscal Module §1-18 completo
2. Identificar tu situación: persona física vs SL viable
3. Verificar si Modelo 720 te aplica (broker extranjero > 50k EUR)
4. Plan calendario fiscal (§15)
5. Si P&L > 50k anual → contratar asesor fiscal especializado (§12)
6. Reportar siempre P&L bruto Y neto after-tax (§14)
7. Tax-loss harvesting en diciembre con regla 2 meses (§3.2)
```

### Si vas a validar el sistema antes de operar
```
1. Leer Scenario Library §1-19
2. Para cada uno de los 12 escenarios históricos:
   - Reconstruir datos del periodo
   - Generar señales con sistema actual
   - Verificar coherencia: ¿el sistema reaccionó como teoría predice?
3. Stress test contra escenarios hipotéticos §14
4. Si DD hipotético > 30% en algún escenario → revisar circuit breakers
5. Material de entrenamiento: drill mensual con escenarios random
```

### Si vas a setup infraestructura de continuidad
```
1. Leer Disaster Recovery §1-9
2. Inversión inicial (~250 EUR + 8h):
   - UPS para router/ordenador
   - Móvil 4G/5G plan
   - Power bank
   - Gestor contraseñas (Bitwarden/1Password)
3. Setup contactos de emergencia (§3)
4. Backups configurados (cloud + local)
5. Drill trimestral programado
6. Kill switch documentado
```

### Si vas a trabajar tu psicología de trading
```
1. Leer Behavioral Finance §1-13 (especialmente §2 sesgos y §8 reglas)
2. Implementar pre-trade y post-trade ritual (§4)
3. Empezar journal emocional diario (§5)
4. Establecer cooldowns escritos (§3.1, §6.2)
5. Programar vacaciones forzadas (§7)
6. Auditoría semanal de las 10 reglas (§8.2)
7. Lectura recomendada: Trading in the Zone, The Daily Trading Coach
```

### Si vas a expandir a Crypto / Commodities
```
PRE-REQUISITO: año 4+ con Capa 1 + 3 estables

1. Leer Cross-Asset §1-17 completo
2. EMPEZAR por Gold (más estable, drivers claros)
3. Acceso via ETFs preferido (GLD, IBIT, etc.)
4. Sizing ultra-reducido inicial (commodities 0.5%, crypto 0.25%)
5. NO leverage en crypto bajo ninguna circunstancia
6. Crypto activación: BTC primero, ETH solo tras 6m BTC estable
7. Tracking SEPARADO del resto del portfolio
```

### Si vas a operar multi-broker
```
PRE-REQUISITO: capital > 100k EUR para que tenga sentido

1. Leer Multi-Broker §1-14
2. Setup por Tier (S/M/L/XL) según capital (§2)
3. Master Sheet OBLIGATORIO desde día 1 (§4)
4. Procedure de transferencia entre brokers documentado (§5)
5. Failover procedure escrito (§8)
6. Asesor fiscal informado (statements multi-broker)
7. Reconciliación semanal (full) y mensual (oficial)
```

### Si tu sistema en vivo está fallando
```
1. NO entrar en pánico ni tomar acción inmediata
2. Esperar 24-48h si pérdida es reciente (anti-emoción)
3. Aplicar triage SA §8.3:
   - Situación A → no acción
   - Situación B → reducir sizing
   - Situación C → parar + auditar
4. Documentar diagnóstico por escrito ANTES de cualquier cambio
5. Cambios solo si Situación C confirmada
```
