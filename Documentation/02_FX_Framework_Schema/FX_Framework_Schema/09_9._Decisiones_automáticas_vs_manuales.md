## 9. Decisiones automáticas vs manuales

### Automatizable (sin intervención humana en estado estable)
```
- Descarga de datos, limpieza, transformaciones
- Cálculo de z-scores y Score_Endo / Exo / Total
- Generación de Señal_FX por par
- Filtro de régimen World View
- Cálculo de position sizing (RPT, Kelly, VolTarget)
- Activación de circuit breakers por DD
- Alertas: CFTC en extremos, β drift > 0.15, slippage outliers
- Métricas Self-Awareness diarias / mensuales
- Logging trade journal (auto-llenar campos cuantitativos)
- Health checks de datos
```

### Manual / decisión humana
```
- Verificación checklist pre-entry técnico (alineación multi-timeframe)
- Validación de setup A+ (contexto cualitativo)
- Ejecución de la orden en el broker (a menos que exista API trading)
- Triage diagnóstico A/B/C ante DD significativo
- Decisión recalibrar / parar / continuar tras alerta
- Evaluación de eventos extraordinarios (geopolítica, eventos black-swan)
- Auditoría trimestral / anual / trianual del sistema
- Post-mortem cualitativo de trades perdedores
- Cambios estructurales al sistema (nuevos drivers, eliminaciones)
- Override discrecional (solo en eventos catastróficos no modelados)
```
