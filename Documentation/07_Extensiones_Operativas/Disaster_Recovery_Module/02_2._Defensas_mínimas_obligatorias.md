## 2. Defensas mínimas obligatorias

### 2.1 Conectividad

```text
☐ Internet doméstico principal (fibra/cable)
☐ Backup móvil 4G/5G con datos suficientes (mín 20 GB / mes)
☐ Hotspot configurado y testeado al menos 1× / mes
☐ Acceso al broker desde móvil verificado funcional
```

### 2.2 Acceso a brokers

```text
☐ App móvil broker instalada y verificada login
☐ Web broker accesible desde navegador en móvil
☐ Teléfono dealing desk del broker guardado en contactos
☐ 2FA codes en app dedicada (Authy, Google Authenticator)
   con backup codes guardados en gestor de contraseñas
☐ Email de recuperación accesible
```

### 2.3 Energía

```text
☐ UPS (Uninterruptible Power Supply) para router + ordenador principal
   → 30-60 minutos de autonomía suficientes para cierres ordenados
   → Coste: 80-150 EUR
☐ Móvil siempre cargado > 30%
☐ Power bank cargado para emergencias prolongadas
```

### 2.4 Backups

```text
☐ Trade journal completo en cloud (Google Drive / Dropbox encriptado)
☐ Statements brokers descargados mensualmente y archivados localmente + cloud
☐ Configuración del sistema (β, parámetros) en git con remote
☐ Credenciales en gestor de contraseñas (Bitwarden / 1Password)
   con master password memorizada NO escrita en cloud
☐ Backup codes 2FA impresos físicamente en lugar seguro
```
