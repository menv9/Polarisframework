## 15. Parámetros Default Cross-Asset

| Parámetro | Commodities | Crypto |
|---|---|---|
| Risk per trade | 0.5-0.75% | 0.25-0.5% |
| Kelly fraction | ¼ | ⅙ |
| Vol target | 8-10% | 4-6% |
| ATR multiplier | 3.0-4.0 | 4.0-5.0 |
| Convicción mínima | 1.3σ | 1.5σ |
| Target (R) | 2-3R | 2-3R + partial 1.5R |
| Stop temporal | 8-12 semanas | 4-8 semanas |
| Max simultáneos | 3 | 2 |
| Notional total subset | ≤ 25% capital | ≤ 10% capital |
| Update β | Anual + alerta drift | Semestral (data corta) |
