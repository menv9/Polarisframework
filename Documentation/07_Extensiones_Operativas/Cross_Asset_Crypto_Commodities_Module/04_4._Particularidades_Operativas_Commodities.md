## 4. Particularidades Operativas Commodities

### 4.1 Contango y Backwardation (CRÍTICO para futuros)

```text
CONTANGO: precio futuro > precio spot
  → Mantener long en futuros causa "roll yield" negativo
  → ETFs como USO sufren contango drag (puede ser 5-15%/año)
  → Ejemplo: WTI 2020 con super-contango destruyó USO

BACKWARDATION: precio futuro < precio spot  
  → Roll yield positivo si long
  → Estructura típica de mercados con tightness (oil 2022, copper a veces)

OPERATIVA:
  Verificar siempre la curva antes de operar futuros largos
  Para holding > 2 meses: considerar contango impacto
  Alternativa: ETFs físicos (GLD para oro) que no tienen problema
              Mining stocks como proxy (Capa 3)
```

### 4.2 Roll de futuros

```text
Futuros expiran. Hay que "rollar" antes de expiración:
  - Cerrar contrato actual (nearby)
  - Abrir contrato siguiente (next month)
  
Coste del roll = bid/ask × 2 + diferencial entre meses

Frecuencia:
  Mensual para WTI, Brent, Natural Gas
  Trimestral para metales (típicamente Mar/Jun/Sep/Dec)

Si no quieres gestionar roll: usar ETFs (con caveat de contango).
```

### 4.3 Costes y márgenes futuros

```text
TÍPICOS (Interactive Brokers):
  Gold (GC):     comisión $2.50 por contrato, margen ~$8,000
  Brent:         comisión $2.50, margen ~$6,000
  Copper (HG):   comisión $2.50, margen ~$5,000
  
Tamaño de contratos:
  Gold:    100 oz × precio = ~$200,000 notional/contrato
  Brent:   1,000 barriles × precio = ~$80,000 notional
  Copper:  25,000 lbs × precio = ~$100,000 notional

Implicación: 1 contrato = exposición sustancial.
Con $50k capital, sizing prudente = 1 contrato máx por commodity.
```
