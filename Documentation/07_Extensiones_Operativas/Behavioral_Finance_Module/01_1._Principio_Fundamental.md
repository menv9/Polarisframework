## 1. Principio Fundamental

```text
EL ENEMIGO NO ES EL MERCADO. ERES TÚ MISMO BAJO ESTRÉS.

Estadísticas:
  - 80% de retail traders pierden dinero (FCA, ESMA datos)
  - El sistema rara vez es el problema — la disciplina sí
  - Pérdidas grandes vienen de overrides discrecionales, no del sistema

Implicación:
  Construir el sistema más perfecto del mundo es inútil
  si no construyes también la disciplina para seguirlo.
```
