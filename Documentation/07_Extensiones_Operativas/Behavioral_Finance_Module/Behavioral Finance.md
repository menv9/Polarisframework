# POLARIS Framework — Behavioral Finance & Trading Psychology Module

> **Módulo del operador humano.** El factor #1 de fracaso de sistemas cuantitativos no es el sistema — es el operador que lo abandona en el peor momento. Los sesgos cognitivos no se eliminan; se gestionan con reglas externas y rituales.
>
> **Filosofía:** "Conozco mi sesgo" NO protege contra él. La única defensa robusta es estructura: reglas escritas, checklist obligatorios, cooldowns forzados, journaling honesto. Este módulo formaliza esa estructura.
