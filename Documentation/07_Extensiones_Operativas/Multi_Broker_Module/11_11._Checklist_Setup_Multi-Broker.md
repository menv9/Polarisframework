## 11. Checklist Setup Multi-Broker

```
ANTES DE OPERAR MULTI-BROKER:
☐ Justificación clara para múltiples brokers (capital > 100k o asset class split)
☐ Cada broker es tier-1 regulado
☐ Master tracking sheet creada y testeada
☐ Procedure de transferencia entre brokers documentado
☐ Asesor fiscal informado (multi-broker)
☐ Modelo 720 contemplado si aplica
☐ Failover procedure escrito
☐ Reconciliación mensual programada

OPERATIVA SEMANAL:
☐ Master sheet actualizado diariamente
☐ Reconciliación semanal con statements
☐ Margin usage monitorizado
☐ Trade journal con broker explícito por trade

OPERATIVA MENSUAL:
☐ Reconciliación completa con statements oficiales
☐ Análisis de costes de cada broker (¿alguno mal?)
☐ Revisión de asignación de capital
☐ Update de procedimientos si necesario
```
