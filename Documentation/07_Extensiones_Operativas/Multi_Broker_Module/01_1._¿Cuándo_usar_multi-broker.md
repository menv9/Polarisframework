## 1. ¿Cuándo usar multi-broker?

### 1.1 Casos donde 1 solo broker es óptimo

```text
✓ Capital < 100k EUR (no necesidad diversificación contraparte)
✓ Solo operas G10 + ETFs (un broker tier-1 lo cubre todo)
✓ Quieres simplicidad operativa máxima
✓ Tracking unificado nativo del broker es suficiente

→ Recomendación: Interactive Brokers (IB) o Saxo Bank.
   IB cubre: FX (G10 + EM via NDF Pro), futuros, equities globales, opciones, crypto via ETFs.
   Un solo statement, un solo P&L, una sola declaración fiscal.
```

### 1.2 Casos donde multi-broker tiene sentido

```text
✓ Capital > 100k → diversificación contraparte
✓ Capa 1 FX + Capa 3 equities con brokers especializados distintos
✓ Crypto direct holding (exchange separado)
✓ Acceso a productos no disponibles en un solo broker
✓ Backup operativo (si broker A falla, broker B sigue)
✓ Optimización de costes (un broker mejor para cada producto)
```

### 1.3 Cuántos brokers tener

```text
Capital < 100k:           1 broker (todo en uno)
Capital 100k-300k:        2 brokers (uno principal + uno backup)
Capital 300k-1M:          2-3 brokers (separación por asset class si conviene)
Capital > 1M:             3+ brokers (diversificación contraparte obligatoria)

NO complicar más de lo necesario.
Cada broker añadido es:
  - Una cuenta más que mantener
  - Un statement más que reconciliar
  - Un riesgo operativo más que gestionar
  - Una potencial complejidad fiscal extra
```
