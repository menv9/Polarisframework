## 12. Parámetros Default Multi-Broker

| Parámetro | Default | Notas |
|---|---|---|
| Min capital para multi-broker | 100k EUR | Bajo eso, 1 broker mejor |
| Max % capital en un broker | 60% | Diversificación |
| Min % capital en un broker | 10% | Si menor, no cumple función |
| Reconciliación frecuencia | Semanal full, mensual oficial | — |
| Master sheet frecuencia update | Diaria | Auto si posible |
| Rebalanceo máx frecuencia | Trimestral | Más frecuente = costes |
| Transferencias entre brokers | < 5 / año | Evitar churning |
| Conversión FX entre cuentas | < 4 / año | En bloque, no por trade |
