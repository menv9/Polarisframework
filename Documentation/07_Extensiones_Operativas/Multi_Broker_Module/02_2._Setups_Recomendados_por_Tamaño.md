## 2. Setups Recomendados por Tamaño

### 2.1 Setup Tier S (capital 50-200k)

```text
1 BROKER PRINCIPAL: Interactive Brokers
  - FX G10 + EM
  - Equities globales
  - Futuros (oro, oil, copper)
  - ETFs (incluido crypto post-2024)
  - Margen unificado
  
PROS:
  - Simplicidad máxima
  - Tracking nativo unificado
  - Comisiones competitivas
  - Regulación tier-1 (multi-jurisdicción)
  
CONS:
  - Concentración total de contraparte
  - Si IB tiene problemas: parón total

→ Aceptable para capitales < 200k (FSCS UK garantiza £85k).
```

### 2.2 Setup Tier M (capital 200k-500k)

```text
BROKER 1 PRINCIPAL: Interactive Brokers
  - 70-80% del capital
  - FX, equities mayoritario, futuros

BROKER 2 SECUNDARIO: Saxo Bank o LMAX
  - 20-30% del capital
  - FX backup + ETFs adicionales
  - Operativo si IB tiene downtime

→ Beneficio: si broker 1 falla, broker 2 puede liquidar emergencias.
   Diversificación contraparte mínima.
```

### 2.3 Setup Tier L (capital 500k-2M)

```text
BROKER 1: Interactive Brokers (40-50% capital)
  - FX y equities core

BROKER 2: Saxo Bank (30-40% capital)
  - FX alternativo + ETFs Europa
  - Backup completo

BROKER 3: Prime Broker / specialist (10-20% capital)
  - LMAX (FX ECN profesional)
  - O exchange crypto específico (Coinbase Institutional)
  - Para productos/precios específicos

→ Diversificación contraparte robusta.
   Cada broker absorbe pérdida potencial limitada.
```

### 2.4 Setup Tier XL (capital > 2M)

```text
4+ BROKERS distribuidos:
  - 2 brokers FX (ej: IB + LMAX)
  - 1-2 brokers equities (IB + custodial bank si trust services)
  - 1 broker futuros (puede ser IB)
  - 1 exchange crypto institucional (Coinbase Prime, Kraken Pro)
  - Cuenta bancaria separada para cash > 200k

Estructura típica:
  Cash buffer:        cuenta bancaria principal + secundaria
  Trading:            dispersado entre brokers
  Cold storage:       parte de crypto si direct holding

→ Empieza a parecer "family office".
   Asesor financiero / wealth manager profesional probable.
```
