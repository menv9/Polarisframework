## 9. Pipeline de Ejecución End-to-End

```
┌──────────────────────────────────────────────────────────────────┐
│  Trade ejecutable (Risk Module)                                   │
│  (size, entry, stop, target, plazo)                              │
└──────────────────────────┬───────────────────────────────────────┘
                           │
                           ▼
┌──────────────────────────────────────────────────────────────────┐
│  CHECK COSTES (§1.4)                                              │
│  Edge esperado vs Coste_total estimado                           │
│  Si Edge_neto < umbral → ABORT                                    │
└──────────────────────────┬───────────────────────────────────────┘
                           │
                           ▼
┌──────────────────────────────────────────────────────────────────┐
│  CHECK CALENDARIO (§8)                                            │
│  Eventos próximos 24h                                             │
│  Si NIVEL 1 inminente → DEFER                                     │
└──────────────────────────┬───────────────────────────────────────┘
                           │
                           ▼
┌──────────────────────────────────────────────────────────────────┐
│  TIMING DE ENTRADA (§3)                                           │
│  Default: London open D+1                                         │
│  Si tamaño grande: scaled entry                                   │
└──────────────────────────┬───────────────────────────────────────┘
                           │
                           ▼
┌──────────────────────────────────────────────────────────────────┐
│  ORDEN ENVIADA AL BROKER                                          │
│  Market entry + OCO (stop-market + target-limit)                  │
│  GTD = horizonte de la señal                                      │
└──────────────────────────┬───────────────────────────────────────┘
                           │
                           ▼
┌──────────────────────────────────────────────────────────────────┐
│  MONITORING DIARIO                                                │
│  - Verificar exits jerarquía §4.1                                 │
│  - Verificar circuit breakers (Risk §6.4)                         │
│  - Re-evaluar señal mensual                                       │
└──────────────────────────┬───────────────────────────────────────┘
                           │
                           ▼
┌──────────────────────────────────────────────────────────────────┐
│  CIERRE                                                           │
│  Logging: precio obj, precio ejec, slippage, costes totales       │
│  Output: P&L neto + atribución a Self-Awareness Module            │
└──────────────────────────────────────────────────────────────────┘
```
