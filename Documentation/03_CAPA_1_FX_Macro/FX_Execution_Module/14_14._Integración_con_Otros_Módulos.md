## 14. Integración con Otros Módulos

```text
INPUT:
  ← FX_Risk_Management_Module: trade ejecutable (size, stop, target, plazo)

OUTPUT:
  → FX_Self_Awareness_Module:  
      - Slippage por trade
      - Coste total realizado
      - P&L neto
      - Atribución de coste a (spread, swap, slippage, comisión, fiscal)
  
  → FX_Risk_Management_Module (loop):
      - Si slippage > 95% percentile durante 5 trades consecutivos
        → revisar broker / horarios / tamaño
      - Si coste real > 130% del estimado
        → re-evaluar viabilidad del par en el sistema
```
