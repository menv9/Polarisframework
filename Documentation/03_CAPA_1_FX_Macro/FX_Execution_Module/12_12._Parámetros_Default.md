## 12. Parámetros Default

| Parámetro | Default | Rango razonable |
|---|---|---|
| Sesión preferida | London open D+1 | London open / NY open |
| Tipo orden entry | Market | Market / Limit / Scaled |
| Tipo orden stop | Stop-Market | Stop-Market only |
| Tipo orden target | Limit | Limit only |
| Spread máximo aceptable | 1.5× típico | 1.2× – 2.0× |
| Coste máx vs edge | 30% | 20% – 40% |
| Blackout NIVEL 1 (FOMC/ECB) | −30min / +60min | −60min / +120min |
| Blackout NIVEL 2 (CPI/NFP) | −30min / +30min | −15min / +60min |
| Slippage máx aceptable (G10) | 3 pips | 2 – 5 pips |
| Re-evaluación señal | Mensual | Quincenal – mensual |
| Comisión máx por lote | $5 | $0 – $10 |
