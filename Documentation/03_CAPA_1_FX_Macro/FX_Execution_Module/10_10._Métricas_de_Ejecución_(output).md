## 10. Métricas de Ejecución (output)

Para alimentar Self-Awareness:

| Métrica | Cálculo | Target / referencia |
|---|---|---|
| Slippage medio por trade (pips) | media \|fill − obj\| | < spread típico |
| Slippage 95th percentile | percentil 95 | < 2× spread típico |
| Coste total / P&L bruto | Σ costes / Σ \|P&L bruto\| | < 30% |
| Spread medio pagado | media spread al fill | ≈ spread típico tabla §2.1 |
| % trades ejecutados vs señalados | trades / señales válidas | > 90% |
| Tiempo medio de ejecución | timestamp fill − timestamp orden | < 5 segundos (auto), < 5 min (manual) |
| Fills parciales | % órdenes con fill incompleto | < 5% |
| Re-quotes / rejections | rate de rechazo del broker | < 1% (si > 5% cambiar broker) |
