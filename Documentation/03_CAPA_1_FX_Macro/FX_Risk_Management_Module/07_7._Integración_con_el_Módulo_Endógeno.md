## 7. Integración con el Módulo Endógeno

### 7.1 Pipeline completo

```text
┌─────────────────────────────────────────────────────────────────┐
│  MÓDULO ENDÓGENO (II.5)                                          │
│  Output: Señal_FX(A/B) + horizonte dominante + |Señal|/σ_señal   │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│  MÓDULO RISK MANAGEMENT (V) — este documento                    │
│                                                                  │
│  1. Convicción (§3.3)        → mult_convicción                  │
│  2. Vol régimen (§2.2)        → mult_vol                         │
│  3. Position sizing (§3.1)    → MIN(RPT, Kelly, VolTarget)      │
│  4. Stops (§4)                → niveles entry/stop/target       │
│  5. Portfolio check (§5)      → ajuste por correlación/exposición│
│  6. Drawdown check (§6.1)     → mult_drawdown                   │
│                                                                  │
│  Pos_final = Pos_base × mult_convicción × mult_vol × mult_DD    │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
                        Trade ejecutable
                  (size, entry, stop, target, plazo)
```

### 7.2 Ejemplo combinado

```text
Inputs:
  Capital:                 100,000 USD
  Señal_FX(EUR/USD):       +1.6·σ_señal (convicción alta)
  Horizonte dominante:     Medio (1-3m)
  Vol EUR/USD régimen:     Normal (P50)
  Drawdown actual:         3% (Normal)
  Posiciones abiertas:     ninguna
  ATR(14) EUR/USD:         65 pips
  Precio EUR/USD:          1.0850
  
Cálculo:
  Stop distance:           2.5 × 65 = 162.5 pips
  Stop price:              1.0850 − 0.01625 = 1.06875
  Risk per trade:          1% = 1,000 USD
  Pos_RPT:                 1,000 / (162.5 × 10) = 0.615 lotes = 61,500 EUR
  Pos_Kelly (¼, p=0.55):   ~5,760 EUR (binding)
  Pos_VolTarget (12% ann): 150,000 EUR
  Pos_base:                MIN = 5,760 EUR
  
  mult_convicción:         1.0 (alta)
  mult_vol:                1.0 (normal)
  mult_DD:                 1.0 (DD < 5%)
  
  Pos_final:               5,760 × 1.0 × 1.0 × 1.0 = 5,760 EUR
  
  Target (2R):             1.0850 + 2 × 0.01625 = 1.11750
  Stop temporal:           12 semanas (horizonte medio)
  
Trade ejecutable:
  Long EUR/USD 5,760 EUR @ 1.0850
  Stop: 1.06875 (162.5 pips, riesgo ~94 USD = 0.094% capital, sub-Kelly)
  Target: 1.11750 (325 pips)
  Cierre forzado: 12 semanas o flip de Señal_FX
```

> **Observación:** Kelly conservador hace que el riesgo efectivo sea menor del 1% nominal. Esto es correcto y deseable para alguien sin track record largo. Ajustar a ½ Kelly tras 50+ trades con métricas estables.
