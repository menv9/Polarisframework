## 1. Principios Fundamentales

### 1.1 Las 4 leyes innegociables

1. **Nunca arriesgar más de lo que puedes perder sin afectar tu capacidad de seguir operando.**
2. **El sizing depende de la volatilidad, no del precio.** Una posición de 100k EUR/USD y 100k USD/TRY tienen riesgo distinto por 50x.
3. **Stop antes que target.** Define dónde estás equivocado **antes** de calcular dónde tomar beneficio.
4. **Drawdown es asimétrico.** Recuperar un −50% requiere +100%. La protección frente a pérdidas grandes vale más que maximizar ganancias.

### 1.2 Inputs requeridos antes de cualquier trade

| Input | Fuente | Frecuencia |
|---|---|---|
| Señal_FX(A/B) | Módulo endógeno | Mensual (re-evaluación) |
| Volatilidad realizada 21d / 63d | Datos de precio | Diaria |
| Implied volatility 1M ATM | OTC FX options | Diaria |
| ATR(14) en pips | Datos de precio | Diaria |
| Precio actual + bid/ask spread | Broker / data feed | Tiempo real |
| Capital disponible | Cuenta | Diaria |
| Posiciones abiertas (todas) | Cuenta | Tiempo real |
| Drawdown actual (%) | P&L histórico | Diaria |

Si falta cualquiera de estos → **no operar**.
