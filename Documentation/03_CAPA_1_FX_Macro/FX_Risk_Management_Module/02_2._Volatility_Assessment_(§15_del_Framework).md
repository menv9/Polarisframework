## 2. Volatility Assessment (§15 del Framework)

### 2.1 Métricas de volatilidad

```text
Realized Vol (rolling 21d, anualizada):
  σ_realized = std(log_returns_diarios) × √252

ATR (Average True Range, 14d):
  TR_t = max(High_t − Low_t, |High_t − Close_t-1|, |Low_t − Close_t-1|)
  ATR_14 = SMA(TR, 14)

Implied Vol (1M ATM):
  Cotizada por brokers FX options (Bloomberg, Reuters, OTC)
  Refleja expectativas forward, no historia
```

### 2.2 Régimen de volatilidad

Clasificar el par actual contra su propia historia 5Y:

| Régimen | Criterio | Ajuste de sizing |
|---|---|---|
| **Low Vol** | Realized 21d < percentil 30 | × 1.0 (full size) |
| **Normal** | P30 ≤ Realized < P70 | × 1.0 |
| **High Vol** | P70 ≤ Realized < P90 | × 0.7 |
| **Extreme Vol** | Realized ≥ P90 | × 0.4 (o no operar) |

### 2.3 Vol implícita vs realizada (signal de cambio)

```text
IV_RV_ratio = IV(1M) / RV(21d)

ratio > 1.3  → mercado espera evento (FOMC, elecciones) → reducir o esperar
ratio < 0.8  → complacencia, posible expansión vol → endurecer stops
0.8-1.3      → régimen estable, sizing normal
```
