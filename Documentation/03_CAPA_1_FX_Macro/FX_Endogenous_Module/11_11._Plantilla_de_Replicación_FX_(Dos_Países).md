### 11.2 Consolidación

```text
Score_corto(A/B)  = Σ Contribución_i  para i ∈ corto
Score_medio(A/B)  = Σ Contribución_i  para i ∈ medio
Score_largo(A/B)  = Σ Contribución_i  para i ∈ largo

Señal_FX(A/B) = 0.20·Score_corto + 0.50·Score_medio + 0.30·Score_largo
```
