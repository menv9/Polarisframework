## 2. Arquitectura Diferencial

### 2.1 Fórmula maestra

```text
Señal_FX(A/B) = Score_FX(A) − Score_FX(B)

donde:
  Score_FX(país)  = Σᵢ ( βᵢ · zᵢ · signoᵢ_FX · filtro_régimenᵢ )
  zᵢ              = z-score 10Y del indicador i
  βᵢ              = peso óptimo calibrado por regresión
  signoᵢ_FX       = +1 / −1 según mapeo FX (sección 4)
  filtro_régimenᵢ = ajuste dinámico por risk-on/off (sección 7)
```

### 2.2 Variables del sistema

| Variable | Significado | Fuente / fórmula |
|---|---|---|
| **Rate** | Valor actual del indicador macro | Dato oficial / mercado |
| **z-score** | Posición vs distribución 10Y | `(rate − μ₁₀ᵧ) / σ₁₀ᵧ`, clip a [−10, +10] |
| **Signo FX** | Dirección esperada del impacto en la divisa | Tabla §4 |
| **β (peso)** | Sensibilidad histórica del FX al indicador | Regresión §9 |
| **Régimen** | Estado del mercado (risk-on / risk-off) | VIX + credit spreads + S&P trend (§7) |
| **Score por horizonte** | Sub-score parcial corto / medio / largo | §6 |
| **Señal final** | Spread `Score_FX(A) − Score_FX(B)` | Salida del modelo |

### 2.3 Interpretación de la señal

```text
Señal_FX(A/B) > 0   →  A se aprecia vs B  (long A/B)
Señal_FX(A/B) < 0   →  B se aprecia vs A  (short A/B)
|Señal| < umbral    →  Sin convicción     (flat)
```

El umbral mínimo de convicción se fija con backtest (§9): típicamente |señal| > 1·σ del distribución histórica de señales.
