## Apéndice A — Matrices de Juicio Experto v1 (Legacy)

Las matrices originales se conservan como **scoring de respaldo cuando no hay serie histórica suficiente** para calcular z-scores (países emergentes con datos < 5 años, o indicadores recientemente publicados). En esos casos, usar la matriz cualitativa v1 con el signo FX v2.
