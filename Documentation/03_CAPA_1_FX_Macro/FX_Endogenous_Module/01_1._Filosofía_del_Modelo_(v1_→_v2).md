## 1. Filosofía del Modelo (v1 → v2)

El modelo original (v1) puntuaba el sesgo endógeno hacia inflación/deflación de **un solo país** mediante **juicio experto** y pesos asignados arbitrariamente. Para usarlo en FX hay que corregir tres problemas de fondo:

| Eje | v1 (inflación país) | v2 (FX direccional) |
|---|---|---|
| **Alcance** | Absoluto, un país | **Diferencial**, dos países |
| **Variable objetivo** | Sesgo inflación/deflación | **Apreciación/depreciación** de la divisa |
| **Asignación de scores** | Juicio experto (umbrales discretos) | **Z-scores estadísticos** sobre ventana 10Y |
| **Pesos** | Inventados (40/10/10/40/10/50) | **Calibrados por regresión** sobre 15-20Y |
| **Signos** | Orientados a inflación | **Reorientados a fortaleza de divisa** |
| **Indicadores** | 17 macro tradicionales | **17 + 7 drivers FX** (tipos reales, CC, NIIP, ToT, breakevens, CFTC, REER) |
| **Horizonte** | Único, mezclado | **3 sub-scores** (1-4w / 1-3m / 3-12m) |
| **Régimen** | Ignorado | **Filtro risk-on / risk-off** dinámico |
| **Validación** | Ninguna | **Backtest obligatorio** con métricas mínimas |
