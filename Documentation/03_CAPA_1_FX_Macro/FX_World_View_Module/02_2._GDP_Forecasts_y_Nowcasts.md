## 2. GDP Forecasts y Nowcasts

### 2.1 Universo de fuentes

| Fuente | Cobertura | Frecuencia | Lag |
|---|---|---|---|
| **IMF World Economic Outlook (WEO)** | Global, todos los países | Semestral (Abr / Oct) + updates | 1 trimestre |
| **OECD Economic Outlook** | Global G20 | Semestral | 1 trimestre |
| **Bloomberg Consensus Survey** | G10 + EM mayores | Mensual | <1 mes |
| **Atlanta Fed GDPNow** | USA (real-time) | Continua, actualizada con cada data release | 0 (nowcast) |
| **NY Fed Nowcast** | USA (real-time) | Semanal | 0 |
| **ECB BMPE** | Eurozone | Trimestral | 1-2 meses |
| **BoJ Tankan** | Japón (encuesta empresarial, proxy de growth) | Trimestral | 1 mes |
| **PBOC Beige Book equivalents** | China | Mensual | 2-4 semanas |

### 2.2 Indicador operativo: divergencia consensus vs nowcast

```text
GDP_gap(país, t) = Nowcast(t) − Consensus_forecast(t)

Si gap > +0.5pp → economía sorprende al alza → divisa fuerte (ceteris paribus)
Si gap < −0.5pp → economía sorprende a la baja → divisa débil
Si |gap| < 0.5pp → en línea, sin información direccional fuerte
```

Esta señal anticipa **revisiones de consenso**, que son lo que mueve mercados de tipos y FX.

### 2.3 Score de crecimiento global agregado

```text
Score_crecimiento_global = w_USA · GDP_gap(USA) 
                         + w_EUR · GDP_gap(EUR) 
                         + w_CHN · GDP_gap(CHN) 
                         + w_JPN · GDP_gap(JPN) 
                         + w_resto · GDP_gap(resto)

Pesos sugeridos (% del PIB global, snapshot 2024):
  USA:    25%
  EUR:    18%
  CHN:    18%
  JPN:    5%
  Resto:  34%

Interpretación:
  > +0.5  → momentum macro global positivo (favor pro-cíclicas)
  < −0.5  → momentum macro global negativo (favor refugios)
  En between → neutro
```

### 2.4 Citi Surprise Index (proxy práctico)

Si construir GDP_gap completo es complejo, el **Citi Economic Surprise Index** es un proxy directo y disponible:

```text
CESI(país)  = sorpresas macro publicadas vs consenso, ponderadas por relevancia
              y decaying con el tiempo (ventana ~3 meses)

Interpretación:
  CESI > +50  → sorpresas positivas dominantes
  CESI > +100 → sobrecalentamiento macro
  CESI < −50  → sorpresas negativas dominantes
  CESI < −100 → desaceleración severa

Disponible para: G10 + EM mayores
Actualización:   diaria
```

**Regla operativa:** usar `CESI(A) − CESI(B)` como input adicional al diferencial Señal_FX(A/B) cuando esté disponible.
