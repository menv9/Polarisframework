## 10. Integración con Otros Módulos

```text
INPUT:
  ← FX_Endogenous_Module: Señal_FX, horizonte dominante, |Señal|/σ_señal
  ← FX_Endogenous_Module: z_CFTC (también input del modelo, reutilizado)

OUTPUT:
  → FX_Risk_Management_Module:
      - Decisión: trade-now / wait-N-days / abort
      - Si trade-now: precio entry recomendado, stop técnico (si aplica), nivel de invalidación
      - Refinamiento sobre stop ATR-based del Risk Module si hay nivel técnico mejor
  
  → FX_Execution_Module:
      - Tipo de entry preferido (Market on Open, Limit pullback, Break confirmation)
      - Precio limit si aplica
  
  → FX_Self_Awareness_Module:
      - Setup classification (A+, regular, marginal)
      - MFE/MAE de cada trade
      - Performance por tipo de setup
```
