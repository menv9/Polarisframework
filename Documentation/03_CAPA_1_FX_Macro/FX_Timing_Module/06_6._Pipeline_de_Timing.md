```
┌─────────────────────────────────────────────────────────────────┐
│  Señal_FX disponible (convicción ≥ media)                        │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│  CHECK 1: Multi-timeframe alignment                              │
│  Tendencia del filtro (W o M) compatible con dirección señal     │
│  NO              SÍ                                              │
│   └─► WAIT       └─► continuar                                   │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│  CHECK 2: CFTC posicionamiento                                   │
│  z_LF < +2 (longs) o > −2 (shorts)                               │
│  Crowded                Limpio                                   │
│   └─► WAIT purga         └─► continuar                           │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│  CHECK 3: Estructura de mercado                                  │
│  Estructura compatible con dirección señal                       │
│  CoS reciente contra tesis    Estructura intacta                 │
│   └─► WAIT nuevo HL/LH        └─► continuar                      │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│  CHECK 4: Setup técnico de entry                                 │
│  Esperar uno de los siguientes:                                  │
│    A. Pullback a 50dma / 100dma + RSI saliendo de extremo        │
│    B. Ruptura confirmada de nivel + retest                       │
│    C. Pin bar / engulfing en nivel fuerte (≥2 confluencias)      │
│    D. Divergencia bull/bear en nivel                             │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│  CHECK 5: Calendario / blackouts                                 │
│  Sin evento NIVEL 1 en 48h                                       │
└──────────────────────────────┬──────────────────────────────────┘
                               │
                               ▼
                    ┌──────────────────────┐
                    │  TRADE TIMING-READY   │
                    │  Pasar a Execution    │
                    │  Module §3 (entry)    │
                    └──────────────────────┘
```
