## 5. Calendario de Eventos

> Cubierto en detalle en [[03_CAPA_1_FX_Macro/FX_Execution_Module/Ejecución|FX Execution Module]] §8 (blackouts NIVEL 1, 2, 3).

### 5.1 Implicación específica para Timing

```text
Si Señal_FX es válida pero hay evento NIVEL 1 en próximas 48h:
  Opción A: ESPERAR — entrar tras evento, una vez vol normalizada.
  Opción B: SCALED — abrir 30% pre-evento, completar 70% post-evento.
  Opción C: ABORT — si convicción es marginal y evento puede flippearla.

Default: Opción A para horizonte medio/largo (la espera es barata).
         Opción B solo si convicción alta y la señal podría
         materializarse exactamente con el evento.
```

### 5.2 Trades intencionales sobre eventos

```text
NO recomendado como parte del sistema base.
Si el sistema decide operarlos: sub-sistema separado (ver Execution §8.3).
```
