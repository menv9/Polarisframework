## 7. Combinación Endo + Exo

### 7.1 Fórmula maestra

```text
Score_FX_Total(país) = (1 − w_exo) · Score_Endogenous(país)
                     +    w_exo    · Score_Exogenous(país)

Señal_FX(A/B) = Score_FX_Total(A) − Score_FX_Total(B)
```

### 7.2 Ejemplo numérico (AUD/USD)

```text
w_exo(AUD) = 0.60, w_exo(USD) = 0.25

Score_Endo(AUD)  = +0.5  (datos australianos en línea)
Score_Endo(USD)  = −0.3  (USA desacelerándose)
Score_Exo(AUD)   = +1.2  (iron ore subiendo, China PMI > 50, copper up)
Score_Exo(USD)   = +0.4  (USD bias bullish leve)

Score_Total(AUD) = 0.40 · 0.5 + 0.60 · 1.2 = 0.20 + 0.72 = 0.92
Score_Total(USD) = 0.75 · (−0.3) + 0.25 · 0.4 = −0.225 + 0.10 = −0.125

Señal_FX(AUD/USD) = 0.92 − (−0.125) = +1.045

Interpretación: long AUD/USD con convicción media-alta.
                Driver dominante: commodities + China (vía exógeno AUD).
```

### 7.3 Ejemplo de divergencia endo-exo

```text
Divergencia frecuente y peligrosa:
  Score_Endo(AUD) = −1.0 (RBA dovish, datos AU malos)
  Score_Exo(AUD)  = +1.5 (iron ore en máximos, China estímulo)

Score_Total(AUD) = 0.40 · (−1.0) + 0.60 · 1.5 = −0.40 + 0.90 = +0.50

Resultado: el exógeno domina y la señal sigue siendo long AUD,
aunque la fundamental local sea negativa.

Esto es CORRECTO en commodity currencies — el commodity manda.
```
