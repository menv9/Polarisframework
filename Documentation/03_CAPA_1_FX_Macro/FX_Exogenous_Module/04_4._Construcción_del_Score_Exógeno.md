## 4. Construcción del Score Exógeno

### 4.1 Pipeline genérico

```text
Para cada divisa X:

1. Identificar drivers exógenos relevantes (§2 o §3).
2. Para cada driver D:
   a. Calcular z-score 10Y de D (aplicar transformación si no estacionario).
   b. Determinar signo del impacto sobre la divisa.
   c. Calcular contribución: z(D) · signo · β_D
3. Score_FX_Exogenous(X) = Σ contribuciones
```

### 4.2 β iniciales por divisa (placeholder)

Suma de β exógenos no se normaliza a 1; se relativiza al peso exógeno declarado en §2:

```text
Score_FX_Total(X) = (1 − w_exo) · Score_Endogenous(X) + w_exo · Score_Exogenous(X)

donde w_exo es el peso exógeno relativo de §2 (USD: 0.25, JPY: 0.50, AUD: 0.60, etc.).
```
