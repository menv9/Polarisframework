## 11. Métricas Cuantitativas Avanzadas (Opcionales)

### 11.1 Deflated Sharpe Ratio (DSR)

```text
Corrige el Sharpe por número de trials probados (multiple testing).

DSR = SR_observado · sqrt((1 − γ_3·SR + ((γ_4−1)/4)·SR²) / (T−1))

donde:
  γ_3, γ_4 = skewness, kurtosis
  T = número de observaciones

Si DSR > 0.95 (significancia 95%) → Sharpe es real.
Si DSR < 0.50 → probablemente sobreajuste.

Crítico si has probado muchas variaciones del modelo.
```

### 11.2 Probabilistic Sharpe Ratio (PSR)

```text
Probabilidad de que Sharpe verdadero > umbral (típicamente 0).

PSR(SR*) = Φ((SR_obs − SR*) · √(T−1) / √(1 − γ_3·SR + ((γ_4−1)/4)·SR²))

Target: PSR(0) > 0.95 → Sharpe positivo es real con 95% confianza.
```

### 11.3 K-Ratio

```text
Mide la "linealidad" del crecimiento del equity.

K = slope(log_equity) · √n / std_error(slope)

Mayor K = curva más limpia, drawdowns menores en relativo.
Útil para comparar dos sistemas con Sharpe similar pero curvas distintas.
```

### 11.4 Pain Index / Ulcer Index

```text
Pain  = mean(DD_t)              (DD diario promedio)
Ulcer = sqrt(mean(DD_t²))       (RMS, penaliza más DDs grandes)

Útiles para evaluar el "dolor" psicológico de operar el sistema.
Sistemas con Sharpe similar pero Ulcer < son preferibles.
```
