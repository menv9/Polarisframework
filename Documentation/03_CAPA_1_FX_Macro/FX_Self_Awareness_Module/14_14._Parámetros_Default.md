## 14. Parámetros Default

| Parámetro | Default |
|---|---|
| Ventana Sharpe rolling primario | 12 meses |
| Ventana Sharpe rolling rápido | 6 meses |
| Ventana Hit rate rolling | 20 trades |
| Frecuencia post-mortem mensual | día 1 cada mes |

| Frecuencia post-mortem anual | día 1 enero |
| Frecuencia trianual | enero cada 3 años |

| Min sample para decisión estratégica | 100 trades |
