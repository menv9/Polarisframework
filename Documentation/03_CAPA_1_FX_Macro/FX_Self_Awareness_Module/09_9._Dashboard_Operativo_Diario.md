```text
- Equity actual y vs peak
- Drawdown actual + duración
- Métricas rolling (12m, 6m, 3m, 1m)
- Posiciones abiertas y exposure neta
- World View state actualizado
- Eventos próximos 7 días
- Health checks de datos pasados
- Alertas activas
```
