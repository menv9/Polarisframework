## 2. Métricas de P&L (§20)

### 2.1 Sharpe Ratio

```text
Sharpe = (R_anual − R_libre_riesgo) / σ_anual

donde:
  R_anual          = retorno anualizado (geometric mean)
  R_libre_riesgo   = T-bill 3M anualizado (puede asumirse 0 para FX puro)
  σ_anual          = std(retornos diarios) × √252

Cálculos:
  Daily Sharpe annualized = mean(daily_returns) / std(daily_returns) × √252
  Monthly Sharpe annualized = mean(monthly) / std(monthly) × √12

Interpretación:
  Sharpe < 0.5  → no operable (o sistema roto)
  0.5 – 1.0     → marginal, aceptable como punto de partida
  1.0 – 1.5     → bueno
  1.5 – 2.0     → muy bueno
  > 2.0         → excelente, verificar sobreajuste
  > 3.0         → casi imposible sin sobreajuste, AUDITAR
```

### 2.2 Sortino Ratio

```text
Sortino = (R_anual − R_libre_riesgo) / σ_downside_anual

donde:
  σ_downside_anual = std(retornos negativos) × √252
  
Solo penaliza la volatilidad bajista, no la alcista (que es buena).

Interpretación:
  Sortino > Sharpe siempre (matemáticamente)
  Ratio Sortino/Sharpe > 1.5 → distribución asimétrica favorable
  Ratio Sortino/Sharpe ≈ 1.0 → distribución simétrica
  Target Sortino > 1.5 OOS
```

### 2.3 Calmar Ratio

```text
Calmar = R_anual / |Max_Drawdown|

Mide retorno por unidad de drawdown máximo (el peor pico-a-valle).

Interpretación:
  Calmar < 0.3  → sistema con DD desproporcionado al retorno
  0.3 – 0.5     → aceptable
  0.5 – 1.0     → bueno (target mínimo)
  > 1.0         → excelente
  > 2.0         → muy alto, verificar período corto
```

### 2.4 MAR Ratio

```text
MAR = CAGR / |Max_Drawdown|
    ≈ Calmar pero usando CAGR en lugar de retorno aritmético

Más conservador que Calmar para horizontes largos.
```

### 2.5 Maximum Drawdown

```text
MDD = max{ over t1 < t2 } [ (peak − valley) / peak ]

Cálculo rolling:
  Para cada día t: equity_high_to_date(t)
  DD_t = (equity_high_to_date − equity_t) / equity_high_to_date
  MDD = max(DD_t) sobre toda la historia

Métricas relacionadas:
  - Time underwater: días desde último peak
  - Recovery time: días desde valley hasta nuevo peak
  - Pain Index: media de drawdowns diarios
  - Ulcer Index: RMS de drawdowns (penaliza más DDs grandes)
```

### 2.6 Tabla de targets

| Métrica | Sistema operable | Sistema bueno | Sistema excelente | Auditar (sospechoso) |
|---|---|---|---|---|
| Sharpe (OOS) | > 0.5 | > 1.0 | > 1.5 | > 3.0 |
| Sortino (OOS) | > 0.7 | > 1.5 | > 2.0 | > 4.0 |
| Calmar | > 0.3 | > 0.5 | > 1.0 | > 2.5 |
| MDD | < 25% | < 20% | < 15% | < 5% (sospecho) |
| Recovery time | < 12 meses | < 6 meses | < 3 meses | inmediato (sospechoso) |
