## 12. Integración con Otros Módulos

```text
INPUT (consume outputs de TODOS los demás módulos):
  ← Endogenous: Señal_FX, atribución por driver, β actuales
  ← Exogenous: contribución exógena por divisa
  ← World View: régimen actual, transiciones
  ← Timing: setup classification, MFE/MAE
  ← Risk Management: sizing, stops aplicados, exposición
  ← Execution: slippage, costes, latencia, fills

OUTPUT (a TODOS los demás módulos):
  → Endogenous: triggers de recalibración β, indicadores a revisar
  → Exogenous: ajustes de β exógenos, drivers a re-evaluar
  → World View: validación de filtros de régimen
  → Timing: hit rate por setup → ajustar selección de setups
  → Risk Management: ajustes de sizing por convicción / régimen
  → Execution: brokers a cambiar si slippage alto, blackouts a ajustar

LOOP DE FEEDBACK:
  Performance pobre → diagnóstico de cuál bloque falla → cambio focalizado
  → backtest → paper trading → deploy o revertir
```
