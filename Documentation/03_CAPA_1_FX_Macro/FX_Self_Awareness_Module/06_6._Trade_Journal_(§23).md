## 6. Trade Journal (§23)

### 6.1 Template por trade (registro pre-cierre)

```text
TRADE ID: [auto-incrementado]
==============================
FECHA APERTURA:        YYYY-MM-DD HH:MM (timestamp señal vs ejecución)
PAR:                   EUR/USD
DIRECCIÓN:             LONG / SHORT
HORIZONTE:             corto / medio / largo
TAMAÑO:                XX,XXX EUR (X.XX% del capital)

INPUTS:
  Señal_FX:            +X.XXσ  (convicción: alta/media/baja)
  Score_Endo:          +X.XX
  Score_Exo:           +X.XX
  WorldView régimen:   ON / OFF / MIXTO
  z_LF (CFTC):         +X.XX
  Setup técnico:       Setup #X / MoO
  Régimen vol:         Low / Normal / High

NIVELES:
  Entry:               1.XXXX (precio objetivo)
  Entry ejecutado:     1.XXXX (precio real)
  Slippage:            X pips
  Stop:                1.XXXX (XX pips, X.X × ATR)
  Target:              1.XXXX (XX pips, X.X R)
  Stop temporal:       YYYY-MM-DD

TESIS (200 palabras max):
  Por qué esta dirección, qué espero que pase, qué horizonte.
  Qué eventos podrían invalidar la tesis.
```

### 6.2 Template post-cierre

```text
FECHA CIERRE:          YYYY-MM-DD HH:MM
RAZÓN CIERRE:          STOP / TARGET / SIGNAL_FLIP / TEMPORAL / CIRCUIT_BREAKER
PRECIO CIERRE:         1.XXXX
P&L BRUTO:             $XXX
COSTES:
  Spread:              $XX
  Swap acumulado:      $XX
  Comisión:            $XX
  Total:               $XX
P&L NETO:              $XXX
R-multiple:            X.X R
MFE:                   XX pips (max favorable excursion)
MAE:                   XX pips (max adverse excursion)
DURACIÓN:              X días

REVISIÓN (post-mortem corto):
  ¿La tesis se materializó como esperaba?      SÍ / NO / parcial
  ¿El timing fue óptimo?                       SÍ / NO
  ¿Hubo override discrecional?                 SÍ / NO (si SÍ: detallar)
  Lección clave:
    [una frase máximo]
```

### 6.3 Reglas del journal

```text
1. SIEMPRE pre-cierre completado ANTES de ejecutar la orden.
2. SIEMPRE post-cierre completado dentro de 24h del cierre.
3. NO editar pre-cierre ex-post (excepto erratas obvias).
4. Una "lección clave" por trade, no más, no menos.
5. Si lección se repite 3+ veces → escalada a post-mortem mensual.
6. Lecciones agregadas mensualmente: si un patrón emerge, regla del sistema.
```
