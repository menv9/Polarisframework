## 2. España — Marco General (Residente Fiscal)

### 2.1 Tipología de instrumentos y su tratamiento

| Instrumento | Tipo de renta | Tipo aplicable 2024 |
|---|---|---|
| **FX spot directo** | Ganancia patrimonial | Escala 19-28% |
| **CFDs sobre FX** | Ganancia patrimonial | Escala 19-28% |
| **Futuros FX (CME)** | Ganancia patrimonial | Escala 19-28% |
| **Acciones (ETFs y individuales)** | Ganancia patrimonial | Escala 19-28% |
| **Bonos (cupones)** | Rendimiento capital mobiliario | Escala 19-28% |
| **Bonos (ganancia trading)** | Ganancia patrimonial | Escala 19-28% |
| **Dividendos** | Rendimiento capital mobiliario | Escala 19-28% |
| **Opciones (premium)** | Variable según ejercicio | Generalmente ganancia patrimonial |
| **Spread betting** | NO disponible en España (banned) | — |

### 2.2 Escala de ganancias patrimoniales 2024 (residente España)

```text
Hasta   6,000 EUR:        19%
6,000 - 50,000 EUR:       21%
50,000 - 200,000 EUR:     23%
200,000 - 300,000 EUR:    27%
> 300,000 EUR:            28%

Implicación operativa:
  P&L anual ≤ 6,000 EUR → marginal 19%
  P&L anual 50,000 EUR → mezcla 19% + 21% = ~21% blended
  P&L anual 100,000 EUR → mezcla 19% + 21% + 23% = ~22% blended
  P&L anual 250,000 EUR → ~24% blended
```

### 2.3 Diferencia ganancia patrimonial vs rendimiento capital mobiliario

```text
GANANCIA PATRIMONIAL:
  - Trading FX, equities, futuros, CFDs
  - Compensable con pérdidas patrimoniales
  - Saldos negativos arrastrables 4 años

RENDIMIENTO CAPITAL MOBILIARIO:
  - Cupones de bonos, dividendos, intereses cuentas
  - Compensable parcialmente con ganancias patrimoniales (25% máx)
  - Mismas escalas de tipos

Implicación:
  Si tu sistema genera mayoritariamente ganancia patrimonial,
  las pérdidas dentro de patrimonial se compensan 100%.
  Si mezclas (long bonds carry + FX trading), la compensación es solo parcial.
```
