## 15. Aplicación al Sistema

### 15.1 Validation pre-live

```text
Antes de operar dinero real, simular cada escenario sobre el sistema configurado:

PARA CADA ESCENARIO:
  1. Reconstruir datos del periodo
  2. Generar señales con el sistema actual
  3. Simular trades, sizing, stops
  4. Calcular P&L hipotético
  5. Verificar: ¿el sistema reaccionó como espera teoría?

Si en cualquier escenario:
  - DD hipotético > 30% → revisar circuit breakers
  - P&L sistemáticamente perdedor → revisar lógica del régimen
  - Sin señal cuando "obviamente" había → revisar drivers
```

### 15.2 Reentrenamiento del operador

```text
Aleatoriamente, hacer "drill" mensual:
  - Sin mirar los resultados, leer un escenario
  - Predecir qué hubiera hecho el sistema
  - Comparar con lo documentado
  - Identificar gaps en comprensión

Mejora preparación para el próximo evento similar.
```
