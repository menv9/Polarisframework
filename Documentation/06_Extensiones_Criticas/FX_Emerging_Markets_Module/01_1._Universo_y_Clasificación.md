## 1. Universo y Clasificación

### 1.1 Tipología de divisas EM

```text
TIER 1 — EM Líquidas Floating (similar a G10 con caveats):
  MXN, BRL, ZAR, KRW, TWD, PLN
  → Aplican G10 framework + ajustes EM
  → Datos disponibles, libre flotación, mercado profundo

TIER 2 — EM con Fricción / Managed:
  CNY (managed float), INR (capital controls), TRY (intervention)
  → G10 framework con modificaciones grandes
  → Banco central interviene activamente
  → Data points pueden ser manipulados o retrasados

TIER 3 — Pegged / Crawling Peg:
  HKD (peg USD 7.75-7.85), AED (peg USD 3.6725), SAR (peg USD 3.75), DKK (peg EUR)
  → No operar direccionalmente con framework macro
  → Solo operar break del peg (eventos raros, alto riesgo asymmetric)

TIER 4 — Hiperinflacionarias / Crisis Crónica:
  ARS, TRY (transitando), VES, LBP
  → Z-scores 10Y se rompen
  → Modelo macro tradicional no aplica
  → Operar solo via NDFs o evitar
```
