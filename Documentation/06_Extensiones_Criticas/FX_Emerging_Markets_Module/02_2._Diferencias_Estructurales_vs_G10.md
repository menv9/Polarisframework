## 2. Diferencias Estructurales vs G10

### 2.1 Drivers que cambian de peso o aparecen

| Driver | G10 | EM | Razón |
|---|---|---|---|
| Diferencial tipos reales 2Y | β = 0.14 (dominante) | β = 0.10 (importante pero no dominante) | Tipos reales menos arbitrables por capital controls |
| **CDS soberano 5Y** | No usado | **β = 0.18 (NUEVO, dominante)** | Riesgo de default supera a tipos en EM |
| **EMBI Global Spread** | No usado | **β = 0.12** | Sentiment EM agregado |
| **Reservas FX / cobertura imports** | No usado | **β = 0.08** | Solvencia exterior crítica en EM |
| **DXY / USD strong** | β = 0.05 (USD bias) | **β = 0.20 (driver dominante!)** | EM colapsan en USD strong (2014-2016, 2022) |
| Cuenta corriente | β = 0.07 | **β = 0.12** | Vulnerabilidad EM mayor |
| **Política / risk premium idiosincrático** | No modelado | **β = 0.10 (variable)** | Eventos políticos dominan en EM |
| ToT (terms of trade) | β = 0.06 (commodity G10) | β = 0.12 (commodity EM) | Mayor dependencia |
| Carry trade flows (CFTC + EPFR) | β = 0.04 | β = 0.10 | EM como target de carry |
| Inflación local | β = 0.04 | β = 0.10 | Variabilidad mucho mayor |

### 2.2 Drivers G10 que se simplifican o desaparecen en EM

```text
PMIs locales:           datos pobres / poco frecuentes / poco fiables → β reducido
Building permits:       no aplicable o no publicado → eliminar
NMI Services:           no existe en muchos EM → eliminar o usar proxy
Breakevens 5Y5Y:        mercado de inflation linkers no líquido → solo BRL, MXN, ZAR
REER:                   usable pero menos predictivo (intervenciones distorsionan)
```

### 2.3 Pesos exógenos relativos en EM (vs §2 Exogenous G10)

```text
G10 promedio: w_exo entre 0.25 (USD) y 0.60 (AUD)
EM:           w_exo entre 0.55 (CNY managed) y 0.80 (TRY, ARS)

Justificación: en EM, el sentiment global y flujos de capital
dominan sobre cualquier fundamental local.

Tabla EM:
  CNY:   w_exo = 0.55 (managed amortigua)
  KRW:   w_exo = 0.65 (semiconductores + China + USD)
  MXN:   w_exo = 0.65 (US ISM + petróleo + USD)
  PLN:   w_exo = 0.60 (Eurozone + USD)
  INR:   w_exo = 0.65 (Brent + USD + intervention)
  BRL:   w_exo = 0.70 (commodities + EMBI + USD)
  ZAR:   w_exo = 0.70 (oro + EMBI + USD)
  TRY:   w_exo = 0.80 (CDS + USD + idiosincrático)
  IDR:   w_exo = 0.70
  ARS:   No modelar con framework — crisis crónica
```
