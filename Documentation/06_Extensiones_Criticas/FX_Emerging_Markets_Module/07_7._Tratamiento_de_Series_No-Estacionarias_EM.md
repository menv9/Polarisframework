## 7. Tratamiento de Series No-Estacionarias EM

```text
Inflación local extrema (TRY, ARS):
  z-score 10Y se rompe (μ y σ contaminados por régimen)
  Solución: usar ventana 5Y para datos de inflación, tipos
            o detrend HP filter agresivo

Reservas FX:
  Series suelen tener tendencia (acumulación gradual)
  Usar diferencia anual o YoY % como input

Deuda / GDP:
  Igual que G10: diferencia anual, no nivel

Currency redenominations / cambios de régimen:
  Argentina: 4 redenominaciones desde 1970
  → datos pre-redenominación no usables sin ajuste
  → ventana efectiva puede ser solo 5-7 años

Datos sospechosos (manipulación):
  China oficial PMI vs Caixin (privado): usar Caixin
  Argentina INDEC pre-2015: descartar (datos manipulados)
  Turquía CPI post-2018: cuestionado, considerar private estimates
```
