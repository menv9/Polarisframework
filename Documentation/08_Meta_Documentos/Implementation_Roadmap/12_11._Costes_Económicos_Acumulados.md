## 11. Costes Económicos Acumulados

### Costes de infraestructura

```
FASE A (preparación):
  UPS, móvil, power bank, gestor pwd:    ~200 EUR único
  Asesor fiscal consulta inicial:        ~500-1,000 EUR
  
FASES B-D (construcción):
  Hosting cloud (opcional):              0 (local) o 15 EUR/mes
  APIs gratuitas:                        0
  Libros, formación:                     ~200-500 EUR
  
FASES E-H (operativa):
  Cloud hosting (recomendado para vivo): 15-30 EUR/mes = 180-360 EUR/año
  Asesor fiscal anual:                   500-1,500 EUR/año
  Backups, herramientas:                 50-100 EUR/año
  
TOTAL COSTE INFRA AÑO 1:                 ~800-1,500 EUR
TOTAL COSTE INFRA AÑO 2+:                ~700-2,000 EUR/año
```

### Costes de trading reales

```
Spreads + comisiones + slippage:        ~5-15% del P&L bruto típicamente
Swap negativo si carry desfavorable:     variable
Impuestos (España):                      19-28% del P&L positivo

P&L neto-net realista:
  P&L bruto × 0.85 (costes) × 0.78 (tax) = ~66% del bruto
  
Si Sharpe bruto 0.8: Sharpe net 0.5-0.6 aprox.
```

### ROI esperado vs alternativas

```
SISTEMA TARGET (año 3+):
  Sharpe net OOS:                        0.5-0.7
  Retorno esperado anual:                8-15% (con vol 12-15%)
  MDD esperado:                          10-20%
  
ALTERNATIVAS:
  SPY buy-and-hold:                      Sharpe ~0.5-0.6, retorno ~10%/año, MDD ~30%
  60/40 SPY/AGG:                         Sharpe ~0.6-0.7, retorno ~7-8%/año, MDD ~20%
  Hedge fund FX promedio (BarclayHedge): Sharpe ~0.4, retorno ~5-7%/año

CONCLUSIÓN HONESTA:
  El sistema bien implementado podría modestamente superar el 60/40 pasivo
  con vol similar. NO esperar outperformance dramática.
  El valor real es DIVERSIFICACIÓN (correlación baja con equities)
  y CONTROL sobre tu propio riesgo.

Si comparas vs 60/40 puro:
  Edge esperado anual:                   1-3% si todo va bien
  Tiempo invertido:                      400-700 h/año en setup, 100-200 h/año mantenimiento
  → Coste-oportunidad del tiempo es real. Decidir conscientemente.
```
