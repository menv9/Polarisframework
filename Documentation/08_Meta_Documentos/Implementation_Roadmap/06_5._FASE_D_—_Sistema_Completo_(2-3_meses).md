## 5. FASE D — Sistema Completo (2-3 meses)

### Objetivo
Añadir todos los módulos restantes para tener sistema end-to-end.

### Mes 8: Risk Management y Execution

```
SEMANA 29-30:
  ☐ Sizing implementado completo:
    - Pos_RPT con risk per trade configurable
    - Pos_Kelly con p,b del backtest real
    - Pos_VolTarget
    - MIN combinación
  ☐ Multiplicadores: convicción, vol régimen, DD
  ☐ Stops: ATR + temporal + signal flip

  ☐ Portfolio risk:
    - Exposición agregada por divisa

  ☐ Circuit breakers DD niveles
  ☐ Capital ramp-up tracking
  ☐ Health check pre-trade obligatorio
```

```

  ☐ Score exógeno por divisa (G10):
```
