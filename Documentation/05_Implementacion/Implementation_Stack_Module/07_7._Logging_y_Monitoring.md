## 7. Logging y Monitoring

### 7.1 Logging estructurado

```python
from loguru import logger

logger.add(
    "logs/framework_{time}.log",
    rotation="1 day",
    retention="90 days",
    format="{time} | {level} | {module}:{line} | {message}",
    level="INFO"
)

# Uso
logger.info("Daily pipeline started")
```
