## 12. Ruta de Implementación Recomendada

### Fase A — Mínimo viable (4-6 semanas)

```text
Objetivo: pipeline diario que genera Señal_FX para G10 con datos reales.

Hitos:
  ☐ Setup repo + Python env
  ☐ Fetcher FRED + yfinance funcionando
  ☐ Z-score 10Y rolling implementado
  ☐ β placeholders cargados desde YAML
  ☐ Score_Endo + Score_Exo + Señal_FX
  ☐ World View básico (régimen ON/OFF/MIXTO)
  ☐ Logging y alertas básicas
  ☐ Dashboard Streamlit con métricas core
  ☐ Tests unitarios > 60% coverage
```

### Fase B — Calibración real (4-8 semanas)

```text
Objetivo: β calibrados estadísticamente, no placeholders.

Hitos:
  ☐ Regresión OLS con statsmodels
  ☐ Tests anti-espuria §17 endo (p-value, VIF, estabilidad, OOS)
  ☐ Walk-forward con embargo
  ☐ Drivers que pasan tests → set final
  ☐ β nuevos en YAML con metadata (fecha calibración, métricas)
  ☐ Backtest formal con métricas mínimas (§9.4 endo)
  ☐ Comparación de métodos β: anual / rolling / Kalman
```

### Fase C — Risk + Execution (3-4 semanas)

```text
Objetivo: trades ejecutables con sizing y stops correctos.

Hitos:
  ☐ Position sizing (Kelly + RPT + VolTarget)
  ☐ Stops ATR-based + temporales + por flip de señal
  ☐ Portfolio risk: exposición neta por divisa
  ☐ Circuit breakers (DD niveles)
  ☐ Capital ramp-up tracking
  ☐ Cálculo de costes (spread + swap + slippage)
  ☐ Integración con broker (paper trading primero)
```

### Fase D — Self-Awareness (2-3 semanas)

```text
Objetivo: feedback loop funcionando.

Hitos:
  ☐ Métricas P&L (Sharpe, Sortino, Calmar, MDD)
  ☐ Métricas proceso (Hit, PF, IC, Expectancy)
  ☐ Atribución por bloque y driver
  ☐ Trade journal con templates
  ☐ Triage A/B/C diagnóstico
  ☐ Dashboard completo
  ☐ Alertas configuradas
```

### Fase E — Capa 2 y 3 (3-4 semanas cada una, año 2+)

```text
Objetivo: añadir capas adicionales solo cuando Capa 1 esté validada.

Hitos Capa 2 FX Trend:
  ☐ Setups técnicos (C2-1, C2-2, C2-3)
  ☐ Kalman Filter para β
  ☐ Sizing reducido + reglas de no doble exposure
  ☐ Tracking separado en Self-Awareness

Hitos Capa 3 Equities:
  ☐ Universo ETFs implementado
  ☐ Mapeo driver → instrumento
  ☐ Score_Equity por instrumento
  ☐ Broker equities integrado (puede ser otro)
  ☐ Tracking separado
```
