## 11. Deployment

### 11.1 Setup local (arranque)

```bash
# Instalación inicial
git clone <repo>
cd fx_framework
python3.11 -m venv venv
source venv/bin/activate    # Linux/Mac
# venv\Scripts\activate     # Windows
poetry install

# Configuración
cp .env.example .env
# editar .env con API keys

# Test
pytest tests/

# Primera descarga de datos
python scripts/initial_data_load.py

# Calibración inicial
python scripts/initial_calibration.py

# Daily pipeline manual (verificación)
python scripts/daily_update.py

# Dashboard
streamlit run scripts/dashboard.py
```

### 11.2 Producción (servidor cloud)

```text
PROVEEDORES RECOMENDADOS:
  DigitalOcean droplet (~$6-12/mes para 1-2 GB RAM)
  AWS EC2 t3.small ($15/mes)
  Linode 2GB ($12/mes)

CONFIGURACIÓN:
  Ubuntu 22.04 LTS
  Python 3.11 vía pyenv
  systemd service para scheduler
  nginx reverse proxy si dashboard accesible externamente
  Let's Encrypt SSL
  fail2ban + ufw firewall
  Backups diarios a S3 / Backblaze B2

MONITORING:
  UptimeRobot (gratis hasta 50 monitors)
  Disk space alerts
  Memory usage alerts
  Process alive checks
```
