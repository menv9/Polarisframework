## 6. Scheduling y Automatización

### 6.1 Cron jobs recomendados

```text
DIARIO (16:00 GMT, post NY close):
  scripts/daily_update.py
  - Descarga datos
  - Health check
  - Recalcula scores
  - Genera señales
  - Actualiza dashboard
  - Envía alertas si triggers

SEMANAL (Lunes 09:00 GMT):
  scripts/weekly_review.py
  - CFTC report (publicado viernes)
  - Re-evaluar timing setups
  - Update métricas rolling

MENSUAL (día 1 del mes):
  scripts/monthly_recalibrate.py
  - Rolling β (Capa 1) actualización
  - Métricas Self-Awareness mensuales
  - Post-mortem template generado
  - Comparación vs backtest

TRIMESTRAL (primer día del trimestre):
  scripts/quarterly_audit.py
  - Reclasificación divisas (β vs S&P)
  - Comparación modelo vivo vs congelado
  - Stress test portfolio actual

ANUAL (1 enero):
  scripts/annual_recalibration.py
  - Recalibración formal de β por regresión
  - Walk-forward OOS
  - Auditoría supuestos económicos
```

### 6.2 Implementación con APScheduler

```python
# Ejemplo conceptual
from apscheduler.schedulers.blocking import BlockingScheduler

scheduler = BlockingScheduler()

scheduler.add_job(daily_pipeline, 'cron', hour=16, minute=0)
scheduler.add_job(weekly_review, 'cron', day_of_week='mon', hour=9)
scheduler.add_job(monthly_recalibrate, 'cron', day=1, hour=10)
scheduler.add_job(quarterly_audit, 'cron', day=1, month='1,4,7,10', hour=11)

scheduler.start()
```
