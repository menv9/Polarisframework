# POLARIS FRAMEWORK

> *La estrella polar de la navegación macro sistemática.*

> **Propósito:** marco integral para operar Forex de forma sistemática, replicable y rentable. Cubre desde la lectura macro global hasta la auto-evaluación de la operativa, pasando por análisis por divisa, timing, ejecución y gestión de riesgo.
>
> **Filosofía:** top-down. Macro → Divisa → Timing → Ejecución → Riesgo → Aprendizaje. Cada capa filtra a la siguiente. Saltarse capas (típico error retail: empezar por charts) lleva a operar contra la propia tesis macro.
>
> **Nombre:** Polaris — referencia direccional fija en un universo de ruido macro. El sistema no predice el futuro; orienta decisiones.
>
> **Estado:** v1.4 — framework conceptualmente completo, pendiente de implementación.
