## Mapa de decisión: ¿cuándo upgrade?

```
¿yfinance falla permanentemente y no hay workaround en 48h?
  └── SÍ → Contrata Twelve Data ($8/mes), NO Alpha Vantage Premium.
          (Twelve Data cubre FX y ETFs; Alpha Vantage no aporta macro)

¿Necesitas datos intradía para execution de alta frecuencia?
  └── SÍ → Reconsidera Twelve Data + broker API (IBKR es gratis)
          Pero tu horizonte es 1-3 meses... probablemente NO.

¿Capital en cuenta real > $100k y sistema validado 12+ meses?
  └── SÍ → Evalúa Bloomberg (pero solo si el ROI del dato justifica el coste)

¿Operas para terceros o capital > $1M?
  └── SÍ → Entonces hablamos de infraestructura institucional.
```
